# プロジェクトレベル指示書テンプレート

このディレクトリには、新規プロジェクト作成時のデフォルト指示書が含まれています。

## 指示書の継承

各指示書は以下の優先順位で適用されます：

1. **プロジェクト固有の指示** - 最優先
2. **このテンプレート** - プロジェクト共通のデフォルト
3. **システムレベル指示** - 基本的な動作定義

## カスタマイズ方法

### 全プロジェクトのデフォルトを変更
このディレクトリの `worker.md` を編集

### 特定プロジェクトのみ変更
`projects/<project-name>/instructions/roles/worker*.md` を編集

## テンプレート変数

- `{{WORKER_NUMBER}}` - worker番号
- `{{PROJECT_NAME}}` - プロジェクト名
- `{{CREATED_DATE}}` - 作成日
- `{{WORKER_COUNT}}` - 総worker数
